# Code style

Indent: 4 spaces

```java
public class SomeClass {
    private Object iVar1;
    protected Object iVar2;
    private static final Object VAL3 = val3;
    
    public SomeClass() {
        iVar1 = val1;
        iVar2 = val2;
    }

    public void someMethod(Object arg) {
        doSomething();
    }

    public Object getVar1() {
        return iVar1;
    }

    public void setVar1(Object val) {
        iVar1 = val;
    }
}
```
